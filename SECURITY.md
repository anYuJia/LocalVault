# Security Policy / 安全策略

## Supported Versions

Only the latest stable release of better-douyin is supported with security updates.

## Reporting a Vulnerability / 反馈安全漏洞

We take the security of better-douyin seriously. If you find a security vulnerability, please do not disclose it publicly. Instead, report it privately using GitHub's private vulnerability reporting feature under the **Security** tab of this repository.

我们非常重视 better-douyin 的安全性。如果您发现安全漏洞，请勿公开披露。请使用本仓库 **Security** 标签页下的 GitHub 私有漏洞报告功能进行私密反馈。
